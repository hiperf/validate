# Getting started

## Installation

Install ``@hiperf/validate`` with your favorite package manager:

::: code-group

```sh [npm]
npm i @hiperf/validate
```

```sh [yarn]
yarn add @hiperf/validate
```

```sh [pnpm]
pnpm i @hiperf/validate
```

```sh [bun]
bun add @hiperf/validate
```

:::
